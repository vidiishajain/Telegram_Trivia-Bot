# User Stories

> **Worked example** for the "agent idea" web app. (Stage 3: success as UX.)

1. As a **curious person**, I want to enter a problem + domain of life and get a **concrete
   agent idea** for it, so I know what I could build.
2. As a **visual thinker**, I want a **diagram** of how the agent would work, so I grasp it
   at a glance.
3. As someone impatient, I want to **see live progress** ("researching… writing… drawing…"),
   so a 30-second wait doesn't feel broken.
4. As a **sharer**, I want a **permanent URL** for the result, so I can send it to a friend.
5. As the **owner**, I want the deployed app **behind a password**, so it isn't open to the
   world (or my API bill).

## What "good" feels like

I type one sentence, watch it think in real time, and land on a clean page: a titled idea, a
diagram, a short write-up, a "here's where to start," and the sources it used — at a link I
can share.

## Out of scope

Accounts, editing/regenerating an idea, browsing others' ideas. One idea per submission, each
at its own URL.
