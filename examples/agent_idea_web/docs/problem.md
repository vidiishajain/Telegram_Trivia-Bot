# Problem

> **Worked example** for the "agent idea" web app. (Stage 1: limits of current agency.)

People constantly notice problems in some **domain of their life** — caring for a newborn,
staying social on a remote team, learning music by ear — but don't know what an AI agent
could realistically *do* about it, or where to begin. The gap is between a felt problem and
a concrete, visual, shareable starting point.

Doing this by hand means research + synthesis + sketching a plan — slow, and most people
won't bother. The limit: **turning "here's a problem in this domain" into a researched,
illustrated, buildable agent concept you can share.**

## Why an agent

It chains things a person finds tedious: live web research with sources, a synthesized
write-up, *and* a generated diagram — then saves it as a permanent shareable page. That's
several capabilities composed, which is exactly what this starter is for.
