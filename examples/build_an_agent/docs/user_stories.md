# User Stories

> **Worked example** for the terminal idea generator. (Stage 3: success as UX.)

1. As a **beginner**, I want to type a rough interest ("cooking", "music for kids") and get
   **one concrete agent idea** I could build, so I have a clear first project.
2. As a **beginner**, I want the idea to be **fun and motivating**, not a dry feature list,
   so I actually want to build it.
3. As a **beginner**, I want it **scoped to this starter's tools** (LLM, media, storage, db),
   so it's realistically buildable here.
4. As a **beginner**, I want the idea **saved to a file**, so I can revisit and refine it.

## What "good" feels like

I type one line, wait a few seconds watching it "research" and "design", and a crisp,
named idea appears with a clear first step — and I think *"oh, I actually want to build that."*

## Out of scope (for now)

Multiple ideas at once; refining/chatting about the idea; accounts or history. One good idea,
saved. That's it.
